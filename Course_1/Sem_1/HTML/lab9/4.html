В каскадных таблицах стилей макет сетки CSS или сетка CSS создает сложные адаптивные макеты веб-дизайна более легко и согласованно в разных браузерах.
Существуют и другие методы управления методами компоновки веб-страниц, такие как таблицы, модель boxи CSS flex box. Сетка CSS в настоящее время не является официальным стандартом.
Сетка CSS может создавать больше асимметричных макетов, чем предыдущая сетка и параметры макета.
Это контрастирует с использованием конкретных взломов браузера или сложных обходных кушек. Одна из проблем с использованием плавающих токов в CSS заключается в том, что если контент добавляется в одну часть страницы, это может нарушить поток страницы.
